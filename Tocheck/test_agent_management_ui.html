🚀 How to Use

📝 Step-by-Step Guide:

  1. Navigate to Document Chat: Go to the document chat interface
  2. Switch to Agent Mode: Click "🤖 Agent Chat" button
  3. Open Agent Manager: Click "⚙️ Manage Agents" button
  4. View Agents: See all your custom and predefined agents
  5. Delete Agent: Click the "⋮" menu on a custom agent → Delete
  6. Confirm Deletion: Review the agent details and confirm
  7. Verification: Agent is removed from the list

⚠️ Important Notes:

  • 🗑️ All agents can be deleted
    Both custom and predefined agents can be removed
  • ⚠️ Deletion is permanent
    Deleted agents cannot be recovered
  • 🚨 Predefined agent warning
    Deleting predefined agents affects all users
  • 🔄 Auto-refresh
    Agent list updates automatically after changes
