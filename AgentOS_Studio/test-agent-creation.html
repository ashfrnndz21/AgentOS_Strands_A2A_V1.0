Agent Creation Flow Test

To test the agent creation flow:

  1. Start the development server: npm run dev
  2. Navigate to the Command Centre page
  3. Click "Create Agent" button
  4. Test navigation through all 6 steps:
    • Step 1: Model Selection
    • Step 2: Role Selection
    • Step 3: Memory Configuration
    • Step 4: Tools & Guardrails
    • Step 5: MCP Tools Selection
    • Step 6: Final Submission
  5. Verify that the "Next" button works on each step
  6. Verify that the "Back" button works
  7. Verify that the final submission works
