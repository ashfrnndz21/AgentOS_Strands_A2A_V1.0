🎯 How to Access Individual Agent Analytics

Step 1: You're Currently Here ❌

You're viewing the "Analytics" tab which shows aggregated data:

  • Total Agents: 2
  • Total Conversations: 1
  • Success Rate: 100%
  • Total Tokens: 166

This is NOT individual agent analytics!

Step 2: Click "Agents (1)" Tab ✅

Look at the top of the page and click: Agents (1)

This will switch from aggregated analytics to individual agent cards.

Step 3: Scroll Down to Find Agent Cards ✅

After clicking "Agents (1)", scroll down to see individual agent cards.

Look for a card that says:

✨ Heath Agent
Powered by official Strands SDK

💬 Chat with Agent 📊 🗑️

Step 4: Click the Blue Analytics Button ✅

On the Heath Agent card, click the blue 📊 button.

This will open the individual agent analytics with 4 tabs:

  • Overview: Agent info + key metrics for Heath Agent only
  • Performance: Execution stats for Heath Agent only
  • Tools: Tool usage for Heath Agent only
  • History: Recent conversations for Heath Agent only

🔍 Quick Visual Guide

Current View (Wrong):

📊 Analytics Tab → Shows aggregated data across all agents

Correct View (What You Want):

👥 Agents Tab → Individual agent cards → 📊 Analytics button → Individual analytics
